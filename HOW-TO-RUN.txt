NEW IN THIS VERSION (v3):
  🎤 Voice Assistant in AI Chatbot
      • Click the 🎤 mic button to speak your question
      • AI reads replies aloud (toggle Auto-speak ON)
      • Click 🔊 on any message to hear it read
      • Works in Chrome/Edge (best support)

  🔔 Full Notification System
      • Bell icon (🔔) in sidebar header with unread badge
      • Dropdown panel — filter All / Unread, mark read, delete
      • Dedicated Notifications page (sidebar → 🔔)
      • Students get: course alerts, streak reminders, placement news
      • Teachers get: at-risk alerts, analytics reports, award reminders
      • Awarding points to a student auto-sends them a notification

  Previous features still included:
  ✅ YouTube thumbnails on course cards
  ✅ Course progress tracking (Watchlist / In Progress / Done)
  ✅ Star ratings, grid/list view toggle
  ✅ Toast notification system
  ✅ Dashboard alert banners
  ✅ Study streak tracker

QUICK START

  Terminal 1 (Backend):
    cd edupredict-enhanced/backend
    npm install
    node server.js
    → Runs at http://localhost:3001

  Terminal 2 (Frontend):
    cd edupredict-enhanced/frontend
    npm install
    npm run dev
    → Open http://localhost:5173

NOTE: Frontend works standalone without the backend.
